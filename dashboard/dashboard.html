XRay Scoring Reference

Formal specification of all metrics, formulas, and classification rules.

1

Composite Score

Eq. 1
S = z(W) × 1 + z(V) × 3 + z(R) × 2 + log10(E) + Sb
where:
S= composite score (dimensionless, centered near 0)
z(x)= (x − μ) / max(σ, 0.001) — z-score normalization
W= Wilson lower bound of engagement rate (see Eq. 2)
V= virality ratio = RT / max(likes, 1)
R= reply rate = replies / max(likes + RT, 1)
E= total engagement = likes + retweets + replies
Sb= save bonus = SR × 2 if bookmarks > 0, else 0
Eq. 2
W = (p̂ + z²/2n − z√(p̂(1−p̂)/n + z²/4n²)) / (1 + z²/n)
where:
W= Wilson score lower bound (95% confidence)
p̂= observed proportion = engagements / views
n= sample size (views)
z= 1.96 (95% confidence z-value)
Ex. 1
Post: 50 likes, 10 RT, 5 replies, 2000 views, 3 bookmarks.

E = 65   ER = 3.25%   W = Wilson(65, 2000) = 2.60%
V = 0.2   R = 0.083   SR = 0.15%

Batch: μW=2.0, σW=1.5, μV=0.15, σV=0.1, μR=0.05, σR=0.04
z(W) = 0.4   z(V) = 0.5   z(R) = 0.825

S = 0.4 + 1.5 + 1.65 + 1.81 + 0.3 = 5.66

2

Component Metrics

Def. 1
Engagement Rate (ER)
ER = (likes + retweets + replies) / max(views, 1) × 100
Def. 2
Wilson ER
W = Wilson lower bound of ER (Eq. 2). Used in composite score.
Def. 3
Virality (V)
V = retweets / max(likes, 1)

Weight: 3x

Def. 4
Reply Rate (R)
R = replies / max(likes + retweets, 1)

Weight: 2x

Def. 5
Save Rate (SR)
SR = bookmarks / max(views, 1) × 100

Bonus: SR × 2 when bookmarks > 0.

3

Statistical Tests

Def. 6
Cohen's d (Effect Size)
d = (μtop − μbot) / spooled
spooled = √((stop² + sbot²) / 2)

|d| < 0.2 negligible, 0.2–0.5 small, 0.5–0.8 medium, > 0.8 large.

Def. 7
Coefficient of Variation (CV)
CV = (σ / μ) × 100

Score spread relative to mean. CV > 80% = high variance.

Def. 8
Chi-Squared Test (χ²)
χ² = Σ (Oi − Ei)² / Ei

O = observed hook type count in top tier. E = expected from overall distribution. df = types − 1. Critical χ² at p=0.05 looked up by df.

Def. 9
Confidence Interval (95% CI)
CI = μ ± 1.96 × (σ / √n)
